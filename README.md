# Face Recognition System

Ett Python-program för ansiktsigenkänning med ett grafiskt gränssnitt (GUI). Systemet kan registrera nya användare, träna en AI-modell och känna igen ansikten i realtid.

## Funktioner
*   **GUI**: Användarvänligt fönster byggt med Tkinter.
*   **Live Igenkänning**: Visar namn och roll (t.ex. Student/Teacher) direkt på skärmen.
*   **Automatisk Träning**: Tar bilder och uppdaterar modellen automatiskt.
*   **Rollhantering**: Sparar roller i `roles.json`.

## Installation

1.  Se till att du har Python installerat (3.11 rekommenderas).
2.  Installera biblioteken:
    ```bash
    pip install -r requirements.txt
    ```

## Hur man kör

Starta programmet genom att köra `gui_app.py`:

```bash
py gui_app.py
```

## Filer i projektet
*   `gui_app.py`: Huvudprogrammet med fönstret.
*   `face_core.py`: Logiken för igenkänning och databashantering.

# Face Recognition System (English)

A Python based face recognition program with a graphical user interface (GUI). The system can register new users, train an AI model, and recognize faces in real-time.

## Features
*   **GUI**: User-friendly window built with Tkinter.
*   **Live Recognition**: Displays name and role (e.g. Student/Teacher) directly on screen.
*   **Automatic Training**: Captures images and updates the model automatically.
*   **Role Management**: Saves roles in `roles.json`.

## Installation

1.  Ensure you have Python installed (3.11 recommended).
2.  Install the libraries:
    ```bash
    pip install -r requirements.txt
    ```

## How to Run

Start the program by running `gui_app.py`:

```bash
py gui_app.py
```

## Project Files
*   `gui_app.py`: The main program with the window.
*   `face_core.py`: The logic for recognition and database management.
